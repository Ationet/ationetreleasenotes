# ATIONET Console (Web Portal) - v2.6.7800

## PROD Deployment (Mar 25, 2025)

### ⚠️ **Important:** Transaction Visualization Period Limit

Starting with this version, the transactions section will display, by default, information from the **last 3 months**.

To view older transactions, it will be necessary to use the **“Date From”** and **“Date To”** filters, ensuring that the selected range does not exceed **3 months**.

This change affects the **Transactions**, **Rejected Transactions**, **Exceptions**, and all other transaction-related views within the portal. Including Fleet, Consumer Card, Gift Card and Loyalty transactions

### "Custom" Bulk Actions
Starting with this version, a new functionality called **"Custom"** has been added to the following bulk actions:

- From the **vehicle list view**:
  - Assign status
  - Resend Identifier
  - Enable
  - Disable
- From the **driver list view**:
  - Resend Identifier
  - Enable
  - Disable
