# Local Agent v2.6.0001

## Fecha de liberación: Oct 31, 2025

### [Guía para instalar o actualizar a la versión 2.6.0000 o superior](https://github.com/Ationet/ationetdocs/blob/master/Local%20Agent%20Installation%20Guide-ES.md#instalacion-de-version-260000-o-superior)

### Actualización remota
A partir de esta versión, el servicio incorpora un nuevo proceso de actualización remota automática.
Esto permite que, cuando haya una nueva versión disponible, la instalación se realice de manera automática, sin necesidad de intervención manual por parte de los usuarios.

De esta forma, se garantiza que todos los entornos se mantengan actualizados con las últimas mejoras, correcciones y optimizaciones del sistema.

### Pestaña de Monitoreo Remoto
En el portal de ATIONET, se encuentra una pestaña llamada **'Gateway'** la cual permitirá a los administradores de red ver el estado de Local Agent y algunos otros tipos de terminales que soporten la mensajería.

Desde aquí se podrá visualizar el último reporte del servicio o terminal y en caso de que el sistema no haya recibido novedades en más de 5 horas, la misma se mostrara como sin conexión. A su vez, se podrá ver la versión que se tiene instalada, el sitio donde se encuentra y la fecha del último reporte.

### Nuevo parámetro en el archivo de instalación
En el archivo 'install.config' utilizado para instalar el servicio, se encuentra un nuevo parámetro 'Terminal Code'. 
El mismo deberá ser completado con el mismo código de una terminal en ATIONET (Administración -> Terminales / Controladores) y sera utilizado el sisitema para diferenciar cada instancia del servicio.
