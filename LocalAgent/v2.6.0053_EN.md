## Release Date: April 06, 2026

### [Guide to install or upgrade to version 2.6.0000 or higher](https://github.com/Ationet/ationetdocs/blob/master/Local%20Agent%20Installation%20Guide-EN.md#installation-version-260000)

### Quota Rule Start Date
In version v2.6.7900 of the ATIONET console, a new field has been added that allows you to configure the start date for monthly and annual quota rules. This enables you to define the day of the month or year on which each rule begins to be accounted for.

Now, in this Local Agent version, this start date is also taken into account for offline transactions.
