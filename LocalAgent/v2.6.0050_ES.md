# Local Agent v2.6.0050

## Release Date: Feb 11, 2026

### [Guía para instalar o actualizar a la versión 2.6.0000 o superior](https://github.com/Ationet/ationetdocs/blob/master/Local%20Agent%20Installation%20Guide-EN.md#installation-version-260000)

## Nuevas validaciones
- Se incorporan validaciones sobre la fecha de Cese de Operaciones del sitio y si el mismo se encuentra activo o no
- Se mejoran las validaciones realizadas sobre las entidades de la Autorización al momento de procesar la Finalización de Venta para alinear la respuesta con el host

## Versión de Local Updater
Esta actualización se libera junto a la versión 1.0.0005 del Local Agent Updater, el cual incluye mejoras en el proceso de actualización del Local Agent

## Incidencias corregidas
- Error que impedía el correcto funcionamiento del servicio de actualización del Local Agent
- Error en la consola de ATIONET donde el Local Agent no tenía en cuenta la casilla "No soporta Offline" para los programas de Consumer Card y Gift Card.
