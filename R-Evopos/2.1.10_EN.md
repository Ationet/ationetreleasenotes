### Loyalty Discounts in Batch Closure
The number of discount operations performed during the period has been added to loyalty batch closures.

- *Discount* with the number of discount operations
- *Total Discounted* with the total amount of all applied discounts

### Improvements to Loyalty Module Configuration
A new option is now available in the *Modules -> Loyalty* section that allows selecting whether to use Points Community, Discounts, or Both.

<img width="247" height="151" alt="image" src="https://github.com/user-attachments/assets/c886bde2-12ac-41c3-9e64-463822770ff7" />

### Optional Fleet Module
There is now an option to disable the Fleet module. This is because a terminal may operate exclusively with Loyalty and without Fleet, making this module optional.

If the Fleet, Consumer Card, and Gift Card modules are disabled but Loyalty remains active, then the "Operations" and "Administration" sections will be hidden, leaving only the "Loyalty" and "Maintenance" sections available.
